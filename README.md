# b11a11-client-side-mdhossain-2437

High-performance Car Rental System client built with Vite + React, Tailwind CSS, GSAP, Lenis, TanStack Query, React Router, and Firebase.

## Scripts
- `npm run dev` – start dev server
- `npm run build` – production build
- `npm run preview` – preview built app

## Environment
Create `.env` in client:
```
VITE_API_URL=http://localhost:5000
VITE_FIREBASE_API_KEY=
VITE_FIREBASE_AUTH_DOMAIN=
VITE_FIREBASE_PROJECT_ID=
VITE_FIREBASE_STORAGE_BUCKET=
VITE_FIREBASE_MESSAGING_SENDER_ID=
VITE_FIREBASE_APP_ID=
```

## Tech
- React (Vite), Tailwind v3
- GSAP, Lenis
- TanStack Query, Axios (withCredentials)
- React Router
- Firebase Auth (Email/Google)
